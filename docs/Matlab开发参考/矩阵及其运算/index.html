矩阵及其运算

变量与赋值​

在matlab中，数值分为实数和虚数​

real()函数可以提取复数的实数部分，imag()可以提取复数的虚部，complex()可以生成复数

a = complex(2,9)
b = real(a)
c = imag(a)

clf命令可以清空图形​

x = 0:0.1:2*pi
plot(x,sin(x))
clf

figure可以设定图像显示窗口 使用close关闭改窗口

MATLAB常用数学函数​

tip

abs(x) sqrt(x)

取整函数 round(x) fix(x) floor(x) ceil(x)

对数函数 log(x) log2(x) log10(x)

指数函数 exp(x)

矩阵的建立与拆分​

基本函数​

zeros(3,4)
ones(3,4)
rand(3,4)
randn(n)    %% 均值为0，方差为1的标准正态分布 n X n 的随机分布矩阵 Y

矩阵的拆分​

A = [1 2 3; 4 5 6; 7 8 9]
B = A(1:2,:)

抽取矩阵的某一行以及多行​

a = [1 2 3; 4 5 6; 7 8 9]
b = a(1,:)
c = a([1,2],:)

抽取矩阵的某一列以及多列​

a = [1 2 3; 4 5 6; 7 8 9]
b = a(:,1)
c = a(:,[1,2]])

重排与复制​

tip

B = reshape(A,m,n)

B = repmat(A,m,n)

%% 将A复制六行
A = [1 2 3 4]
B = repmat(A, 6, 1)
